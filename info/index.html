What? Why Definomica

on 2024-08-18

Definomica exists because anyone should be able to learn DeFi concepts on the fly as they please.

This is completely opensource and free to use.

You can contribute to the project by submitting a PR or opening an issue on the GitHub repository. Check the contributing guidelines to get started.
